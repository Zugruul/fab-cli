Thousands-strong knowledge corpora (like the card DB) are GENERATED, never minted: a deterministic script owns the note files end-to-end — idempotent rebuild, a check/diff mode for staleness, links.json written by the generator, and a preserved hand-curated section per note for accrued rulings. Mint stays reserved for hand-curated knowledge. Recall seeds on EXACT tag matches, so the generator must engineer the tag surface deliberately (name tokens, categories, numeric facets).
