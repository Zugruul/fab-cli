Pair every such run with (1) an index/hub note linking all generated notes, (2) a link-index rebuild ([[brain-bulk-writes-need-link-reindex]]), and (3) a directory regeneration. Quote definitions verbatim with exact section cites so each note is independently verifiable against the vendored source.
