Property rulings (CR 2.0.3): modifying base value ≠ "gaining/losing" (Big Bully doubling base {p} dodges anti-gain effects, 2.0.3b); modifying non-base = gain/loss (Korshem). Cost property is IMMUTABLE (2.2.4) — reductions apply only in the play-process math; "cost" references = unmodified property, "payment" references = what was paid (2.2.4b). Name matching: exact whole-word case-insensitive (2.7.5 — "Proto" doesn't match "Protos"); moniker ≠ name (2.7.3c). Life totals: discrete permanent modifications, not continuous effects (2.5.3b); base-life change recalculates (2.5.3c); can exceed base (2.5.3d); floor 0 → cleared/dies as game-state action (2.5.3f, [[game-state-actions-and-reversal]]). Metatypes can never be gained/lost (2.6.5); (c)-defined power/defense evaluates 0 when indeterminable (2.3.2a, 2.9.2a). Self-references mean "this", not same-named cards (2.12.3a).
