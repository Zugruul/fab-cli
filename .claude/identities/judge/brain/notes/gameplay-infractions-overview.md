Gameplay infractions violate the Comprehensive Rules and are assumed unintentional (intentional → Cheating, PPG §4.2) (PPG §2). Repeat rule: 3+ of the SAME gameplay infraction in a tournament DAY upgrades that infraction by one severity level (max Game Loss); resets daily. Exception: Failure to Maintain Game State never upgrades for repeats.

The taxonomy (default penalty):
- Game Rules Violation (§2.1, Warning): mis-applying the rules for elements you're responsible for, not covered elsewhere (e.g. underpaying a cost, defending from arsenal). Fix: simple rewind, else partial fix, else full rewind, else continue; upgrade to IP2 if unfixable with significant advantage.
- Failure to Maintain Game State (§2.2, Warning): failing to acknowledge the opponent's gameplay infraction (other than Missed Trigger). Can't be committed alone; never upgrades for repeats.
- Missed Trigger (§2.3, Caution): see the dedicated [[missed-trigger-timing-windows]].
- Looking at Extra Cards (§2.4, Warning): unintentionally seeing cards that haven't changed zone.
- Hidden Card Error (§2.5, Warning): a private-card error not correctable by public info alone (e.g. over-drawing, merging zones).
- Shuffling Error (§2.6, IP2) and Presenting Cards Error (§2.7, Warning): see [[shuffling-and-marked-cards]] and [[start-of-game-procedure]].

Choose fixes from [[fixing-game-states]]; opponent who let it happen may get Failure to Maintain Game State. Tournament-side infractions: [[tournament-infractions-overview]]. — PPG (latest, 2026-06-10).
