NEVER write ban lists, restricted lists, or living-legend statuses into any note — they change constantly and any snapshot here would go stale and mislead a ruling.

For any card-legality question (banned / restricted / suspended / living legend / format-legal), fetch live, every time, from the official Card Legality Policy:
https://fabtcg.com/rules-and-policy-center/card-legality-policy/

The TRP defines the mechanics of legality (restricted = max 1 copy; banned = not allowed; living-legend heroes are illegal in Classic Constructed but legal in Living Legend) in [[official-formats]], but WHICH cards are in each state is only ever the live policy.

CR 9.4.2 similarly points living-legend status to the official resource page (fabtcg.com/living-legend/) rather than fixing it in rules. A decklist containing a currently-banned card is handled as Decklist Error (upgrade to Game Loss), see [[tournament-infractions-overview]]. For card TEXT (not legality) use [[card-text-from-json]]. — TRP §7; fabtcg.com (fetch live).
