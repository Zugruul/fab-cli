**Ash** (Illusionist/Draconic Token Ash): "Material — While Ash is under an object, that object has phantasm." Material (CR 8.4.5) is a label for a while-static ability: "Material - While this is under a permanent, [EFFECTS]." Ash is put under an object as a sub-card (CR 3.0.14).

Ruling: when the top-card (the object Ash is under) is destroyed and moves to the graveyard, CR 3.0.14e governs Ash directly: "If a top-card ceases to exist, its sub-cards are cleared as part of the same event." Clearing (CR 3.0.12/3.0.12a) sends a token sub-card out of existence, not through a separate "destroy" event of its own — Ash never independently triggers a reflexive "when this is destroyed" ability (it has none here, but this generalizes to any Material sub-card token with such text) because it is CLEARED simultaneously with its top-card, not destroyed as its own discrete game event. There is no window where Ash exists alone in the graveyard granting anything, and no separate priority window between the top-card's destruction and Ash's clearing.
