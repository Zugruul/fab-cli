A FAB game is two heroes reducing each other to 0 life (CR 4.5.3a). Each player starts with their hero and equipment/weapon in the arena; hand size equals the hero's intellect {i}.

A turn is three phases in order: Start, Action, End (CR 4.0.3a). Players only get priority in the Action Phase (CR 1.11.3), see [[priority-and-the-stack]].
- Action Phase: turn-player gets 1 action point (CR 4.3.2). One action card per turn unless it has go again, which refunds an action point on resolution (learn-to-play; CR 8.3.5).
- Pay costs by pitching cards from hand: pitch value is the colour strip — blue 3, yellow 2, red 1 resource (learn-to-play). Pitched cards go to the bottom of the deck at end of turn in the player's chosen order (CR 4.4.3c).
- End-of-turn procedure (CR 4.4.3): reset ally life, optionally arsenal one card, bottom the pitch, untap, lose action/resource points, then draw up to intellect. On the FIRST turn only, all players draw up to intellect (CR 4.4.3f).

Cards are only played, never pitched or used to defend, from the arsenal (learn-to-play). Combat is its own sub-loop: [[combat-chain-steps]]. — CR (latest, 2026-06-10).
