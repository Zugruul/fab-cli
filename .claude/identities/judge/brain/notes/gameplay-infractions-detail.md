Full gameplay infraction catalog with defaults (PPG 2, deepens [[gameplay-infractions-overview]]). Meta-rule: 3+ same-type gameplay infractions in one day → upgrade 1 level (max Game Loss), resets daily; never applies to FtMGS (PPG 2.0). — GAME RULES VIOLATION (2.1, Warning): wrong rules application not covered elsewhere; fix = immediate → simple rewind; later → partial fix, else full rewind, else play on; upgrade IP2 if unfixable + significant advantage; opponent who could have caught it gets FtMGS Warning (or GRV if shared responsibility, e.g. playing around opponent's prevention effect). — FAILURE TO MAINTAIN GAME STATE (2.2, Warning): never exists alone, never upgraded for repeats. — MISSED TRIGGER (2.3, Caution): relevancy windows: decision-triggers → before controller passes priority; rules-affecting → before an illegal action; visible-on-resolution → before actions that presuppose resolution; invisible → before first visible effect. No infraction if no impact, or optional-only (assumed declined); partially-resolved trigger missed = GRV instead; cascading triggers resolve in sequence; upgrade to Warning if advantage + self-created; opponents may stay silent but can't ENGINEER a miss (that's [[conduct-infractions-detail]]). — LOOKING AT EXTRA CARDS (2.4, Warning): fixes by deck state — randomized → shuffle in; known non-randomized → leave; unknown → opponent tops/bottoms them; 5+ from non-randomized deck → IP2 + shuffle. — HIDDEN CARD ERROR (2.5, Warning): private cards inseparable by public info; opponent splits the collection back (deck cards: opponent tops/bottoms); illegal face-down equip → removed from game; missing cards from collection → reveal rest + IP2. — SHUFFLING ERROR (2.6, IP2 default!): illegally randomizing an ordered set (shuffling pitch into deck, absent-minded mid-game shuffle). — PRESENTING CARDS ERROR (2.7, Warning): legal list/pool, illegal presentation; fix composition, opponent resolves ambiguous starting sets; flip-card wrong face → downgrade Caution; transcend-face unfixable → IP2; pro REL → IP2; post-fix deck off by 2+ → Game Loss (off-by-one: no upgrade).
