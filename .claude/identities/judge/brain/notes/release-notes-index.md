HUB: official per-set release notes digested into this brain, Uprising → Omens of the Third Age (older sets not yet ingested: Everfest, Tales of Aria, Monarch, Crucible of War, Arcane Rising, Welcome to Rathe). SUPERSESSION RULE: newer release notes + the current CR OVERRIDE older set pages — some interactions have CHANGED since printing. Known changes to watch: [[release-notes-high-seas]] — [[kw-spectra]] no longer force-closes the chain (closes only when no legal targets remain) and the end phase gained an untap step; [[release-notes-compendium-of-rathe]] — vs ally attacks the defending hero is now the ally's controller (wagers work), equipment defending an attacking ally establishes that controller as attacking hero, and face-down arena objects now KEEP identity + counters; [[release-notes-outsiders]] carries the 2023-03-15 errata (Back Heel Kick, Give and Take, bottom-of-OWNER's-deck); [[release-notes-bright-lights]] carries name errata (Qi Unbound → Qi Unleashed, Chivarly → Chivalry). When an older note conflicts with a newer one, the newer set page + current CR win — verify in third_party/fab-rules/en-fab-cr.txt. CROSS-SET RECURRING PRINCIPLES (asked constantly): (1) conditional continuous effects turning on DURING/AFTER the damage step apply their bonus but damage is NEVER recalculated; (2) go again must exist by the link/resolution step — no retroactive action points; late-granted "when this attacks/hits" abilities never retro-trigger; (3) NO priority during start phase, end phase, close step, or between effects of one resolving trigger/charm/mode bundle; (4) "play without paying {r}" or alt-cost-0 = cost set to 0 THEN increases apply (Frostbite still taxes); cost DEcreases apply AFTER increases; (5) destroying cards in deck/banished triggers NO destroy-watchers (nobody controls them); (6) ward/prevention auras' leave-arena triggers stack AFTER the damage event finalizes — never help against the damage that destroyed them; (7) "can't be prevented" lets replacements do their non-prevention parts (Ward still self-destroys), and prevented "next/first time" triggers are still consumed; (8) open combat chains use Last Known Information for departed attacks.
