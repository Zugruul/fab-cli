Tournament infractions violate the TRP and are assumed unintentional (intentional → Cheating, PPG §4.2) (PPG §3). If a tournament-rule breach isn't listed here, educate/correct but don't penalize; ignoring the instruction is Unsporting Conduct. Repeat rule: 2+ of the SAME type in a day upgrades one level (max Game Loss); resets daily. Exceptions: Tardiness and Slow Play don't follow the repeat-upgrade.

Taxonomy (default penalty):
- Tardiness (§3.1, IP2): not at the table when the timer starts / decklist not submitted. <1 min → Warning; >10 min → Game Loss.
- Note Taking and Outside Assistance (§3.2, Warning; →Game Loss if strategic advantage) — see [[note-taking-rules]].
- Slow Play (§3.3, Caution) — see [[slow-play-vs-stalling]].
- Decklist Error (§3.4, IP2) and Card-Pool Contents Error (§3.5, IP2) — see [[decklists-registration]] and [[deck-checks]].
- Draft Procedure Violation (§3.6, Warning minor / Game Loss major).
- Marked Cards (§3.7, Warning) and Insufficient Shuffling (§3.8, Warning) — see [[shuffling-and-marked-cards]].
- Communicating Incorrect Game Information (§3.9, Warning): mis-stating visible/public info (or not fully answering) when the opponent acts on it — rewind to before the affected action.

Tardiness and Decklist Error don't need Head-Judge pre-confirmation despite being IP2 (§1.2). Conduct-side: [[conduct-infractions-overview]]. — PPG (latest, 2026-06-10).
