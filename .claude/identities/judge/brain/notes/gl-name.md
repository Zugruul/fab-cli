**Name** (CR Glossary): effect: An effect keyword that means the player determines the name of an object to be used by another effect.[8.5.21] property: A property of an object. Names are used by the specialization ability and for identifying tokens. Names and pitch values are used to determine uniqueness of a card.[2.7] Keyword note: [[kw-name]]. Index: [[glossary-index]].
