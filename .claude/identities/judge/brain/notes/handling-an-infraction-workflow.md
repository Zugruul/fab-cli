Workflow for handling an infraction (PPG §1.2):
1. Identify which infraction occurred, then follow its section.
2. Don't intervene in an ongoing game unless an infraction occurred or you're investigating. Don't intervene for a Missed Trigger that's only a Caution, or a Caution/Warning infraction the players mutually resolve in a timely manner.
3. If issuing a penalty, explain the infraction, the procedure, and the penalty to all players involved.
4. Confirm any IP2-or-higher penalty with the Head Judge FIRST — except Tardiness and Decklist Error (§1.2). See [[head-judge-authority]].
5. Multiple related infractions: apply each procedure separately, but issue only the single HIGHEST penalty (not stacked).
6. Time extension: if a ruling takes more than a minute, extend the round for the involved players, rounded up to the minute — see [[round-structure-and-timing]].

If a judge makes a mistake: acknowledge, apologize, fix if possible; a judge error that led to a player's violation is a possible reason for the HJ to deviate (§1.2). When policy doesn't fit, the Head Judge may deviate — [[deviations]]. Grounded in [[ppg-philosophy]]; procedures in [[fixing-game-states]]. — PPG (latest, 2026-06-10).
