**Trigger Event** (CR Glossary): A game event that occurs and meets the condition of a triggered effect. Once a trigger event has occurred, it does not need to occur again to satisfy the trigger condition for the triggered effect layer to resolve successfully.[5.5] Index: [[glossary-index]].
