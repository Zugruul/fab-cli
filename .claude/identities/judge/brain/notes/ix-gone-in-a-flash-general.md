GENERAL INTERACTIONS HUB — Gone in a Flash (red Lightning Action–Attack 4{p}/3{d}, 0-cost). TRUE TEXT (Card Vault, CR 2.0.2 authority): "When this attacks, the next time you play an instant card this chain link, you may return this to its owner's hand." Watcher mechanics (verified, see [[ix-fleeing-starbreeze-x-gone-in-a-flash]]): delayed trigger (6.6.3) installed at the "attack" event 7.2.3; fires on PLAY not resolution; its layer stacks ON TOP of your instant (6.6.6) so the return resolves FIRST; ordinal — consumed by your first instant of the link even if declined (6.6.3a/6.6.5f); window = Attack Step through Damage Step (7.0.3d; CR's own Flittering Charge example fixes the "this chain link" window as Attack→Damage Step); return never closes the chain (7.0.3c).

STACK-WINDOW PUMP (human-confirmed 2026-07-12; player-suggested, CR-verified on all four points): play GiaF, and BEFORE passing, respond with targeted pumps (Lightning Press "target attack action card cost ≤1", Flow Through / Livewire Press "target Lightning attack") while GiaF is still a layer. (1) You regain priority after playing it (1.11.5, Layer Step 7.1) — response window exists. (2) Stack objects are targetable (1.8.5a) and an attack-card ON THE STACK is already an attack (1.4.1/1.4.2). (3) NO reset stack→queue→chain: queue is an extension of the stack zone (3.15.7), combat chain is an arena zone (3.1.1), so 3.0.9 never triggers — the pump/on-hit rider persists onto the chain link. (4) The watcher installs only at 7.2.3, AFTER 7.2.2 chain placement and after all players pass (5.3.1a/7.1.2a) — Layer-Step instants can NEVER trip or consume it. Net: pump early without spending the watcher, keep the return for a later utility instant. Cost: pumps committed before defending cards are seen. This does NOT disarm the watcher — from 7.2.3 on, the first instant of the link consumes it.

DERIVED PAIRINGS (cr-verified reasoning from card DB sweep 2026-07-12, not individually protocol'd): TARGETING instants played AFTER 7.2.3 (Lightning Press/Flow Through/Livewire Press/Instant-Auras à la Starbreeze) suffer the LKI trap — take the return and the pump fizzles (1.2.3d); use the stack window instead. NON-TARGETING instants (Blink, Cosmic Suture, Flash Bolt, High Voltage, Cosmic Flare, Cloud Cover, Constella cycle) = clean return. BLINK LOOP: play Blink → return resolves first → Blink grants action point → re-attack the same GiaF, fresh watcher on the new link; each Blink re-buys the 4{p} attack. ELECTROMAGNETIC SOMERSAULT: trips the watcher; if the watcher's return is taken, GiaF is off the active link before EMS resolves and can't be chosen; declining (watcher spent anyway) lets EMS return it at link resolution. NEGATION (Aetherize): watcher fired on PLAY — return still resolves even if the instant is negated; opponent's instants never trip it ("you play"). NON-TRIPS: "as though it were an instant" plays (Eclectic Magnetism, Chain Lightning, Kano, Cindering Foresight — type stays Action, true text demands "instant card"); instant-speed ACTIVATED abilities (equipment/Amulets/hero abilities — not playing a card; exploit: Amulet of Lightning grants go again, Zyggy re-fires a Lightning aura's ETA trigger, all without spending the watcher); DISCARDING instants (Astravolt/Flowshard Elemental, Cap of Quick Thinking, Oscilio CI). Oscilio Forked Continuum PLAYING the discarded instant DOES trip it. Meld Shock halves are instant-typed → trip. Sibling per-link watchers (Beckoning Brilliance, Blast to Oblivion, Flowstate Embodiment) can never coexist on GiaF's link — one attack per link. UNSETTLED (inherited from [[ix-fleeing-starbreeze-x-gone-in-a-flash]], do not rule): whether the hit/damage event still occurs when the attacker returned before the Damage Step (7.5.2b covers only the target) — #ask-a-judge.
