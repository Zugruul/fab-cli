PROXIES (TRP 4.3): only Head-Judge-issued, ever. Valid situations: foiling causes marked cards with no reasonable substitute; damage from limited product / opponent / official; insufficient basics at limited; insufficient created cards despite reasonable effort; Ira welcome decks. Proxy valid ONLY for that tournament; self-made or carried-over proxies are illegal. Original card info must be referenceable in-game. PLACEHOLDERS (4.4): official only, represent DFCs in private zones; must swap to the real card when entering a public zone; checklist cards need exactly one box checked. TOKENS (4.5): unofficial tokens OK if card-like (7:5 ratio, ≥half card, ≤card size, readable, unconfusable); comp/pro REL requires physical tokens for everything your registered cards can create incl. state variants; marker-on-token may represent multiples; no pre-placed tokens before creation.
