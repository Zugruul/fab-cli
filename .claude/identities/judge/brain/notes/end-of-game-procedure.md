When time is called in a timed Swiss round and no winner of the current game is determined (TRP §3.4): the turn-player finishes their turn, then ONE additional turn is played. If, at the moment time expires, the turn-player has no decisions left or has clearly indicated they're ending their turn, their turn is complete and the opponent takes the additional turn. If still no winner after that turn, the game is a DRAW. The match winner is whoever won more games, else a draw.

Elimination rounds cannot draw (§3.4). If a game would draw (stalemate/deadlock/mutual defeat), the judge walks players through the tiebreak steps:
1. Warn that highest life total decides if no progress is made toward a result.
2. Winner = higher current life total; if tied, go to 3.
3. Winner = more games won; if tied, start a 4-turn tiebreaker game (2 turns each), then step 4.
4. Continue the tiebreaker until one hero has a higher life total.
5. If it still draws, winner = higher standing before the match.

At the end of a game players must reveal hidden-info cards (e.g. cloaked equipment) on request (§3.4). Judges guide the procedure; see [[draws-and-concessions]] and [[round-structure-and-timing]]. — TRP (latest, 2026-06-10).
