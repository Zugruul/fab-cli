**Drill Shot** (Ranger Action Arrow Attack): "If Drill Shot has an aim counter, it has piercing 1. When this hits a hero, put a -1{d} counter on an equipment they control." Piercing (CR 8.3.23): "Piercing N is a static ability that means 'If this is defended by an equipment, it gets +N{p}.'"

Ruling: the condition is "defended by an equipment" — it is NOT "defended ONLY by an equipment." If Drill Shot (with an aim counter, so piercing 1 is live) is defended by BOTH a hand card AND an equipment on the same chain link (legal when nothing restricts multiple defenders, e.g. no dominate present on that attack), piercing still applies its +1{p} because an equipment IS among the defending cards — the presence of a non-equipment co-defender does not disqualify the bonus. This contrasts with keywords like Heavy (CR 8.4.17), which explicitly require EXCLUSIVITY ("If this is the only card equipped...") — piercing has no such exclusivity clause.
