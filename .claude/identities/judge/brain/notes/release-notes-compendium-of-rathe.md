Compendium of Rathe (legal 2026-02-13) release-notes rulings. EXPLICIT RULE CHANGES starting with this set (supersede older set pages): (1) if an attack's attack-target is an ALLY, the defending hero = the ally's controller — [[kw-wager]]s "with the defending hero" now work vs ally attacks (Cheating Scoundrel, Descend into Madness); (2) when equipment defends an attacking ally, the ally's controller is established as the ATTACKING hero (Crown of Frozen Thoughts, Two-Faced); (3) objects turned face-down in the arena remain the SAME object and RETAIN counters (Recede to Mistform). New keyword [[kw-sharpen]]: put a +1{p} counter; a delayed trigger removes ALL +1{p} counters at end of turn (even from other sources; other counter types stay); if the counter can't be placed you still "sharpened"; +1{p} and -1{p} counters coexist. [[kw-bond]] reminder: the Earth/Ice/Lightning card must be pitched WHILE paying that card's cost, but the bond stays met if the pitched card later leaves the pitch zone. Rulings: Topsy Turvy — "look at top" ≠ "put on top" (Opt bottoms everything; Seek Horizon costs paid by bottoming still count); Lunar Mirage — one trigger per 6+{p} defender, controller orders, ends as copy of LAST resolved, LKI if defender gone, copying a phantasm card DOES trigger phantasm; Doubling Season doubles +{p} EVENTS (not counters, not set/decrease), works on the stack; Tiger Trap strips +{p} effects from prior links, no damage recalc, LKI for departed attacks, can't touch -{p}/set; base-{p} effects apply before other modifiers; Bear Hug-type play restrictions must be met BEFORE paying (intended pitches don't count); you can [[kw-intimidate]] yourself (no draw), empty-hand intimidate still counts; Graven Gaslight — NO priority mid-damage-event (can't Spellvoid, re-equip, Spellvoid again on one event); Colors of Aria has its talents only face-up (pitching makes it public → bonds yes; but can't be revealed for fusion/type reveals); Vestige of Flagellation catches the first ACTUAL life gain if an earlier replacement ate the first; [[kw-ward]] 0 still destroys, prevents nothing; search may fail-to-find but must shuffle.
