Limited formats (TRP 8): SEALED — limited-pool = opened cards + any number of basics from the set; young hero from the set; ANY number of copies; deck EXACTLY 30; hero locked once registered (changeable between games if no registration) (8.1). Registration: open-and-observe across the table → swap pools → opponent registers → swap back → validate; judge signs corrections (8.1.1). DRAFT — pods of 8, 3 packs, pass left/right/left; deck exactly 30; NO communication/signaling during draft (verbal hints, upside-down cards, note-taking all prohibited, 8.2.1 examples); DFC drafting concealment procedure (draft under the pile); picks final once face-down; no reviewing mid-pack; called drafts mandatory at pro REL; pod-internal pairings; playoff bracket by seating (8.2). BLITZ PRECON: blitz rules, precon pool (8.3). CRACK-SHUFFLE: 3 packs, remove back cards, whole pool = deck, optional re-crack per round (8.4).
