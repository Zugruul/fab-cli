A Missed Trigger (default Caution) is when a player, by inaction, fails to acknowledge a triggered effect THEY CONTROL by the time it becomes relevant (PPG §2.3). Acknowledging = a visible game-state change from the effect, or players communicating that it triggered; any player can acknowledge (PPG §2.3).

Relevancy windows — the trigger is NOT missed if acknowledged before the matching point (PPG §2.3, CR 5.5):
- Requires a decision (target/mode): before the controller next passes priority.
- Affects the rules of the game: before an action is taken that the trigger would make illegal.
- Visible effect / decision on resolution: before any player takes an action only possible after it resolved.
- Non-visible effect: before it first visibly affects the game.

Key rulings:
- Optional trigger missed → assumed declined, no infraction (PPG §2.3; CR 5.14 default decisions).
- Players are NOT required to acknowledge opponents' triggers, but may not intentionally cause them to be missed — that's Rules Sharking (PPG §4.10), see [[conduct-infractions-overview]].
- Intentionally ignoring your own MANDATORY trigger is Cheating (PPG §2.3, §4.2).
Upgrade to Warning only if the player would gain advantage AND created the trigger themselves (PPG §2.3). Connects to [[triggered-effects-and-ordering]] and [[gameplay-infractions-overview]]. — PPG/CR (latest, 2026-06-10).
