VERIFIED INTERACTION (CR build 2026-06-10). Fleeing Starbreeze (blue, Lightning Illusionist Instant–Aura, 0-cost: enters-arena → up to 1 target attack gets go again; Ward 1) × Gone in a Flash (red, Lightning Action–Attack 4{p}/3{d}, 0-cost: when this attacks, next instant YOU play this chain link → may return it to hand). CORE ORDERING: the return watcher is a delayed-trigger (CR 6.6.3, source-independent) installed at the Attack Step; playing Starbreeze (legal window: Attack Step priority through the DAMAGE Step — Resolution Step is too late, 7.0.3d) fires the watcher at 5.1.10, whose layer stacks ON TOP of Starbreeze (6.6.6) — the return decision resolves BEFORE Starbreeze does, and the ordinal watcher is CONSUMED even if declined (6.6.3a/6.6.5f). TRAP: take the return with one Starbreeze → the card is in hand when Starbreeze's trigger targets; LKI isn't a legal target (1.2.3d) → no go again in the LKI snapshot → no action point at 7.6.2a. One copy = return OR go again, never both. BOTH-WORLDS LINE: respond to the return layer with a SECOND go-again grant (e.g. second Starbreeze — it won't re-fire the spent watcher); it resolves first, the attack gains go again, THEN the return resolves — LKI now carries go again → action point pays with the attack back in hand (7.6.2a). Second grant on an attack that already has go again fails (8.3.5c). Returning the attacker NEVER closes the chain (7.0.3c LKI; cf. Electromagnetic Somersault, [[release-notes-rosetta]]). "Up to 1 target": zero is legal (1.12.1b); opponent's attacks and QUEUED attacks (Omens queue, [[release-notes-omens-of-the-third-age]]) are legal targets; go again granted after 7.6.2 is never retroactive. [[kw-ward]] 1: mandatory self-destroy prevention on hero damage; leave-arena consequences stack AFTER the damage event. Starbreeze is a true "Lightning aura" → Omens holo-counter banish/return re-fires its enters-arena trigger as a NEW object. [[kw-starfall]]: playing it feeds nothing (resolves to arena); only its DESTRUCTION (Ward etc.) puts an instant in the graveyard. Opponent's instants and duality-cycle ability activations do NOT trip the watcher ("YOU play an instant CARD"). UNSETTLED (escalated, do not rule): whether the hit/damage event still occurs when the attacker returned to hand BEFORE the Damage Step — CR 7.5.2b covers only the target ceasing; #ask-a-judge.
