Uprising (legal 2022-06-24, 226 cards) release-notes rulings. New mechanics: [[kw-affliction]] (enters under an OPPOSING hero's control; owner unchanged — dies to owner's graveyard); [[kw-invocation]] (DFC resolves flipped into arena as a dragon ally; resets on leaving); [[kw-quell]] X (OPTIONAL pay-X-prevent-X on hero damage; if used, a delayed trigger destroys the source at beginning of the END PHASE — the source works all turn until then; no carryover; inert if damage already 0; destroy trigger created even if prevention disallowed); [[kw-ward]] X (MANDATORY prevent X then self-destroy; same 0-damage/no-carryover rules; destroyed even if prevention disallowed); [[kw-transform]] (create the permanent if missing, put subjects under it; fails with no subjects); [[kw-freeze]] (frozen objects can't be played/activated — incl. arsenal defense reactions; statics still work; frozen equipment can still DEFEND); [[kw-rupture]] (bonus at chain link 4+); Ash [[kw-material]] (grants phantasm from underneath). Dragon on-attack triggers fire in the attack step and resolve even if the dragon ceases to exist ([[kw-phantasm]]) — but not if the chain closes in the layer step. Dromai: playing a red any time before combat resolution retroactively gives the attacking dragon go again; dragon dead before resolution = no action point. Draconic links use Last Known Information on an open chain; cost decreases apply AFTER increases (Rising Resentment stays 0 under Frostbite). Themai: opponents can't play/activate ANYTHING but can defend, pitch, use replacement effects, resolve triggers. Iyslander: a Frostbite created in RESPONSE to a played card does not tax it; Frostbite creation is respondable; actions played as instants stay non-attack actions. Hypothermia strips EFFECT-granted go again retroactively (printed go again survives). Insidious Chill triggers per Ice fuse, resolves BEFORE the fused card; can't respond to your own fuse. Tiger Stripe Shuko "can't be prevented": Ward still self-destroys; prevention amounts not consumed. Double Strike: banished copy is a new object (keeps go again, loses rebanish); the chain link remains (counts for rupture). Semblance kills already-triggered phantasm layers. Frost Hex: each Frostbite ping is a separate arcane damage instance (each takes Arcane Barrier 1). Red Hot: rupture damage is UNTYPED. Erase Face strips class/talent in ALL zones.
