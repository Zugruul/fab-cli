Ability functionality is a top ruling area (CR 1.7.4): default = functional only when source is public in arena. Defending non-permanents: abilities NON-functional unless the ability explicitly functions while defending (activated that says so / trigger on defending / stated static exception) (1.7.4a). Other exceptions: cost-payable-only-from-private-zone activateds (1.7.4b), resolution abilities when resolving (1.7.4c), meta-static outside game (1.7.4d), play-static when played from any zone (1.7.4e), property-static always (Mutated Mass, 1.7.4f), while-static per condition (1.7.4g), zone-move replacement statics (Drone of Brutality, 1.7.4j). Layers outlive sources (1.7.1a). Modal: modes locked at stack-time, evaluated then (1.7.5e — Sacred Art counts blue cards at play time, not resolution). Connected ability pairs: the following ability refers ONLY to its leading ability's events; if nothing to refer to, it fails (1.7.6b).
