Start-of-game (CR 4.1) ruling points: first-turn choice — game 1 random-winner chooses; game N: first player to have lost the previous game chooses; after a draw, same chooser (4.1.3). Equipment + deck selections are private and LOCK at deck-presentation (4.1.7a) — changing after seeing opponent's gear is an infraction ([[gameplay-infractions-overview]]). Card-pool leftovers = private inventory; constraint-violating leftovers are removed from the game (Taylor duplicates, 4.1.6c). End-of-turn procedure order is mandatory (4.4.3a-f: ally life reset → arsenal option → pitch bottoming (hidden order) → untap → lose action/resource points → intellect refill) — misordered execution causes many game-state errors. LOSING mid-game (multiplayer): loser's controlled objects cleared, then ALL their owned objects removed from game — even ones opponents control, like their Frostbite tokens (4.5.1a). Draw taxonomy: simultaneous death, effect, intentional (any time, 4.5.4c — see TRP for tournament reporting), stalemate (can't advance) vs deadlock (won't advance) (4.5.4d-e).
