Effect-grammar ruling ladder (CR 1.8): optional effects require full feasibility unless phrased "may choose to" (1.8.3b — Heave needs pay + arsenal slot). Target vs non-target: "target" = declared at stack time, public-only (1.8.5a-b, same target once per phrase); non-target subjects declared at generation (1.8.5c, Remembrance) — resolves many "when did you have to choose?" calls. Undetermined parameters: instructed player picks at generation; multiple players pick in clockwise order from controller (1.8.6a). Insufficient legal objects → all legal ones used (1.8.6c). Effect FAILURE (1.8.9): fails if all targets illegal/gone, no legal parameters, or every event fails — partial success ≠ failure. Missing property = condition unmet; missing numeric property = 0 only when a value is required from a specific object (1.8.7a). "Counts as"/"as though" modifies only the applicable effect (1.8.8). "Your next attack" attaches when a qualifying attack comes under control, not to already-controlled objects that later qualify (1.8.10a).
