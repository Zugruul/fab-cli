FAB defines five information categories (TRP §5.1). Judges use these to rule what a player must answer and what a judge may tell them:
- Visible: physically represented — card state, zones, printed properties of public cards/tokens, counters, hero life. Players maintain their own representation.
- Public: not physically represented but known — non-printed properties of public objects, current/past public decisions defining state, unspent action/resource points, current turn/phase/combat step, match score.
- Assisted: unrelated to game state — printed properties of ANY FAB card/token, game rules, tournament rules/policy, official tournament info. A judge MUST provide requested assisted information if logistically possible.
- Historical: previously public but no longer affects the current state (e.g. what a now-hidden revealed card was). No one must recall it.
- Private: not knowable by all — printed properties of unseen private cards, private decisions.

Rules (§5.1): players must recall and answer honestly about Visible and Public info, and must point out incorrect Visible/Public info. Players may NOT misrepresent Visible, Public, Assisted, or Historical info, nor misrepresent one type as another. Getting this wrong maps to Communicating Incorrect Game Information (PPG §3.9), see [[tournament-infractions-overview]]. Ties to [[judge-calls-and-appeals]]. — TRP (latest, 2026-06-10).
