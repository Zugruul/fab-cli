Full tournament infraction catalog with defaults (PPG 3, deepens [[tournament-infractions-overview]]). Meta-rule: 2+ same-type in a day → upgrade 1 level (max Game Loss), except Tardiness/Slow Play; unlisted tournament-rule violations → educate, no penalty (ignoring the judge then = Unsporting Conduct). — TARDINESS (3.1, IP2): not seated at round start / procedure not completed; <1 min → Warning; >10 min → Game Loss + drop unless they report; missing decklist = tardy until submitted; extend round time by the delay. — NOTE TAKING & OUTSIDE ASSISTANCE (3.2, Warning): illegal notes/advice during game, draft, or pre-hero-selection registration; destroy the notes; guilty SPECTATOR → Game Loss for THEIR next game + ejection from area; strategic advantage gained → Game Loss. — SLOW PLAY (3.3, Caution): educate, observe 1-2 decisions; repeat → Warning; after two Warnings → Game Loss; judges must OBSERVE it themselves, never award time on accusation alone. — DECKLIST ERROR (3.4, IP2): list ≠ intent or illegal; fix list to intent; minor + self-reported/pre-round-1 → Warning; missing hero with only one reasonable choice → Warning; significant difference or advantage from ambiguity → Game Loss; illegal cards intended for play → Game Loss + remove/replace with Cracked Baubles. — CARD-POOL CONTENTS ERROR (3.5, IP2): pool ≠ list; missing-only → Caution; no-advantage extras → Warning; self-reported same-name extras → Warning; significant → Game Loss. — DRAFT VIOLATION (3.6): minor (reversible: wrong-way pass, late pick) → Warning; major (two-card pick discovered late, revealing picks, announcing plans) → Game Loss issued AFTER the draft ends to avoid influencing it. — MARKED CARDS (3.7, Warning): distinguishable face-down; re-sleeve/replace before next match; Cracked Bauble substitution allowed + decklist update, reversible when replacements found; significant advantage → Game Loss. — INSUFFICIENT SHUFFLING (3.8, Warning): supervise re-shuffle; pile counting alone never suffices. — COMMUNICATING INCORRECT GAME INFO (3.9, Warning): wrong/incomplete public info that AFFECTED an opponent's action (markers and card placement count as communication); rewind to before the affected action; stale-marker case at comp REL → Caution possible; opponent assuming without clarifying = no infraction.
