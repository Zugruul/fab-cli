**Fealty** — token keyword (CR 8.6.31).

A Fealty token is a token with the name "Fealty," the supertype Draconic, the subtype aura, the activated ability "Instant -- Destroy this: The next card you play this turn is Draconic." and the triggered ability "At the beginning of your end phase, if you haven't created a Fealty token or played a Draconic card this turn, destroy this."
