PPG penalties, ascending severity (PPG §1.4):
0. Caution — informal, not recorded; minor infraction where procedure + education suffice.
1. Warning — formal, recorded; repeats of the same/similar may upgrade.
2. Intellect Penalty (IP) — reduces the hero's intellect for a number of turns; the lowest penalty with an actual game effect. Default is IP2 (draw 1 fewer card at the end of each of 2 turns). Issued before a game, it reduces the opening draw but keeps the turn count. Multiple IPs extend the turn count. If equal IPs hit all players in a game simultaneously, record but resume as if none issued.
3. Game Loss — ends the current game as a loss; if between games / after round time, applies to the next game. In a best-of-1, a Game Loss ends the match. Simultaneous double Game Loss = double match loss.
4. Disqualification — removed from the tournament; loses the current match, is dropped, forfeits further prizes (keeps prizes already received). No standings place; players below advance one — unless the DQ is after a cut, then no one moves into the cut. DQ requires the player and head judge disqualification forms; DQ'd players are reviewed by the Player Conduct Committee.

All penalties except Caution are recorded to track repeats. Gameplay/tournament infraction upgrades cap at Game Loss (§1.4). Applying these uses [[fixing-game-states]] and [[handling-an-infraction-workflow]]; philosophy in [[ppg-philosophy]]. — PPG (latest, 2026-06-10).
