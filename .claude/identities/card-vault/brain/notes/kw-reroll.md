**Reroll** — effect keyword (CR 8.5.28).

Reroll is a replacement effect. To reroll dice, those dice are rolled again and their result is taken as if it were the first time they were rolled. 8.5.28a If a die is rerolled, its original result is considered to never have happened. 8.5.28b If an event is modified by two or more reroll effects, those reroll effects occur one by one in the order the event was modified.
