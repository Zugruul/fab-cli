**Put / Return (object)** — effect keyword (CR 8.5.15).

Put/Return is a discrete effect. To put/return an object into/to a zone, move it from its current zone to the specified zone. 8.5.15a If a card would be put on top of a deck and there no other cards in the deck, it would be considered to be put on top of the deck and not on the bottom. The same applies if a card would be put on the bottom of the deck - it is not considered to be put on the top. 8.5.15b If a card would be put X cards from the top of the deck and there are fewer than X cards in the deck, the card is put beneath any cards in that zone. The same applies if a card would be put X cards from the bottom of the deck - it is placed above any cards in that zone. 8.5.15c If two or more cards would be put into the deck zone in the same location of a deck, the player chooses the order in which the cards are put there.[1.8.6d] If the deck is private, the order that the cards put into the deck is hidden information.
