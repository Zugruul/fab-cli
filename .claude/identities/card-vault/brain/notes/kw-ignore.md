**Ignore** — effect keyword (CR 8.5.33).

Ignore is a replacement effect. To ignore an event or part of an event, after it has been completed, it is considered to never have happened. 8.5.33a If an effect is conditional or dependent on the result of an event that is ignored, the ignored part of the event is not considered to have happened. 8.5.33b If there are two or more identical parts of an event and the ignore effect does not specify all of those parts, only the specified parts are ignored. 8.5.33c If an event is modified by both a reroll and ignore effect, the reroll occurs before its results are ignored.
