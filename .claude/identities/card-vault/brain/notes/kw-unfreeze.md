**Unfreeze** — effect keyword (CR 8.5.37).

Unfreeze is a discrete effect. To unfreeze an object, all existing freeze effects no longer apply to that object.[8.5.34] 8.5.37a If an object is unfrozen, all existing freeze effects that apply to an object cease to exist. After the unfreeze event has occurred, the object may be frozen again by new freeze effects. 8.5.37b If unfreeze applies to an object that is not frozen, the effect fails.
