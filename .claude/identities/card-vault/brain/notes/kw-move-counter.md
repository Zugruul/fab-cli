**Move (counter)** — effect keyword (CR 8.5.42).

Move is a discrete effect. To move a counter, remove the counter from its current object and put it onto the specified object.[8.5.14][8.5.16] 8.5.42a If no specified counter exists or can be moved, no counters are removed from any object or put onto the specified object.
