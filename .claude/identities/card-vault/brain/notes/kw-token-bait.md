**Bait** — token keyword (CR 8.6.34).

A Bait token is a token with the name "Bait," the subtype aura, the static ability "You can’t play or activate cards you own." and the activated abilitoes "Action -- Destroy this when the chain link resolves: Attack" and "Once per Turn Attack Reaction -- 0: This gets +1{p} and go again."
