**Create (token)** — effect keyword (CR 8.5.2).

Create is a discrete effect. To create a token, produce the specified token and put it in the arena. 8.5.2a The properties of a created token, specified by the token keyword, are defined in .[8.6] 8.5.2b A token is put into the arena as it is created. Effects that trigger from the token entering the arena trigger as though the token was created first, then entered the arena. If an effect applies to the created token as it is created, it applies to it as it enters the arena. 8.5.2c A token enters the arena under the control of the player instructed to create it. If the token is created in a zone owned by a player, the token enters that zone under the control of that player.
