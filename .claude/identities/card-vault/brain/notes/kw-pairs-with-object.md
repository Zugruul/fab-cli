**Pairs [with OBJECT]** — ability keyword (CR 8.3.26).

Pairs is a static ability that means "Equip this only with an OBJECT.", where OBJECT is 1 or more object identities. 8.3.26a A card with pairs can only be equipped to a zone owned by a player if the player already has an OBJECT equipped, or if an OBJECT is being equipped as part of the same event.
