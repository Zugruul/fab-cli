**Roll** — effect keyword (CR 8.5.18).

Roll is a discrete effect. To roll a die, toss a die to produce a sufficiently random orientation, and then the value of the uppermost die face is used as the result of the roll. 8.5.18a A roll is specified with the number and type of dice to roll. If two or more dice are specified then all dice are rolled and their result is taken simultaneously. 8.5.18b A die used in a roll must have uniquely distinguishable faces, with each face representative of a distinct integer value from one up to, and including, the number of faces on the die; and an equal likelihood that it will land face up.
