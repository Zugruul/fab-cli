**Spectra** — ability keyword (CR 8.3.14).

Spectra is a static ability and a triggered-static ability that respectively mean "This can be attacked." and  "When this becomes the target of an attack, destroy this." 8.3.14a An object with Spectra can be the target of an attack, even if it is not a living object.[1.4.5a] 8.3.14b When an object with Spectra becomes the target of an attack, a triggered-layer is put on the stack. When the triggered-layer resolves, the object with Spectra is destroyed. If there are not other legal attack-targets, the combat chain closes.[7.7.2]
