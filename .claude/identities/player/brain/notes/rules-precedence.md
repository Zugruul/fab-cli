Precedence order (CR 1.0.1–1.0.2): card effects supersede CR rules; tournament rules supersede both. Among effects: a RESTRICTION ("can't") beats a REQUIREMENT ("must if possible"), which beats an ALLOWANCE ("may"). "Only X" = restriction equivalent to "everything else can't" (CR 1.0.2a — e.g. Three of a Kind's "may only play from arsenal" blocks even effects allowing plays from banished). Restrictions never retroactively undo game state (CR 1.0.2b — gaining overpower doesn't remove already-declared defenders). Practical play: check for restrictions FIRST when evaluating a line.
