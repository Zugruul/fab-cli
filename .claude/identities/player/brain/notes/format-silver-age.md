Silver Age (SAGE), which the user calls "Sage", is a young-hero constructed format (fabtcg.com/gameplay-formats/silver-age/). Rules: 1 YOUNG hero; 55-card card-pool; deck EXACTLY 40 cards; up to 2 copies of each unique card; only common, rare, or basic rarity cards (a card counts if it was EVER printed at that rarity); lower starting life than CC; best-of-1, 35 min. It is the newer, accessible, faster constructed format — fewer cards, quicker games. Because it is young-hero only, hero pools differ from CC (young slug vs adult slug — see project CLAUDE.md). There is NO "Sage" class; the name is purely the format. Bans are separate and live: [[card-legality-pointer]]. Compare [[format-classic-constructed]] (adult) and [[format-blitz]] (the other young format). Hub: [[player-brain-map]].
