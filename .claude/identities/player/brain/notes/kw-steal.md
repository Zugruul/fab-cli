**Steal** — effect keyword (CR 8.5.54).

Steal is a discrete effect. To steal a card, the player gains control of the object. 8.5.54a The player that controls the effect is considered to have stolen. 8.5.54b If the object is equipped, the object will be equipped by the new controller as part of the steal event. If the object cannot be equipped this way, the effect fails.
