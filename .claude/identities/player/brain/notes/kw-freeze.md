**Freeze** — effect keyword (CR 8.5.34).

Freeze is a continuous effect. To freeze an object, that object cannot be played and its abilities cannot be activated for the duration of the effect. 8.5.34a The player that controls the effect is considered to have frozen an object, and the object is considered to be frozen. The object can be unfrozen by an effect.[8.5.37] 8.5.34b If a freeze effect already applies to an object, a new freeze effect that would apply to that object does not fail.
