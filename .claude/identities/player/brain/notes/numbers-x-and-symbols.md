Number rules (CR 1.12): all values are integers; unspecified fractions round toward zero (3.5→3, -3.5→-3) (1.12.1a); chosen numbers are non-negative; "up to N" includes 0 (1.12.1b). X is undefined-until-defined: while undefined it counts as 0, but the object still HAS the property (1.12.2a) — once defined it stays defined for that object's lifetime (1.12.2b). * = defined by meta-static/effect, else 0 (Mutated Mass outside a game, 1.12.3a).
