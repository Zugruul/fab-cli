**Give** — effect keyword (CR 8.5.53).

Give is a discrete effect. To give an object to a player, that player gains control the object. 8.5.53a The player that controls the effect is considered to have given. 8.5.53b If the object is equipped, the object will be equipped by the new controller as part of the give event. If the object cannot be equipped this way, the effect fails.
