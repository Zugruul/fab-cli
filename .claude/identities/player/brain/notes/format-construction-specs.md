Deck construction numbers (TRP 7 — the tournament-legal specs behind [[format-classic-constructed]] etc.): CC: 1 adult hero, card-pool ≤80, start deck ≥60, max 3 copies per unique card (name+pitch). Living Legend format: same as CC but LL heroes/signature weapons legal, separate ban list. Blitz: 1 young hero, pool ≤52, deck EXACTLY 40, SINGLETON (max 1 copy). Silver Age ("Sage"): 1 young hero, pool ≤55, deck EXACTLY 40, max 2 copies, everything common/rare rarity only ("ever printed common/basic" counts as common). Restricted = 1 copy max; banned = 0 — always check the live legality policy ([[card-legality-pointer]]). Legendary = max 1; Unlimited = any number ([[kw-legendary]], [[kw-unlimited]]). Card uniqueness = name + pitch ([[names-monikers-classes-talents]]). Links: [[format-silver-age]], [[format-blitz]], [[format-living-legend]], [[game-start-and-inventory]].
