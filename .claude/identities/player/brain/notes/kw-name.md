**Name** — effect keyword (CR 8.5.21).

Name is a discrete effect. To name an card, declare the name property of an card.[2.7] 8.5.21a A card cannot be named using its moniker, the full name of the card must be declared. 8.5.21b Only specified cards that exist in the game can be named. The card does not have to be legal to play in the current game to be named. The player cannot name a card that does not meet the specifications of the effect.
