**Exchange** — effect keyword (CR 8.5.49).

Exchange is a discrete effect. To exchange objects, the objects swap their respective zone, visibility, and control with each other. 8.5.49a The swap in zone, visibility, and control for both objects happens simultaneously as a move event. If either object cannot complete the move successfully, the effect fails. 8.5.49b If an object is equipped, the object that it will be exchanged with will be equipped as part of the move event. If the object cannot be equipped this way, the effect fails.
