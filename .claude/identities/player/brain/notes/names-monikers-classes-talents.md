NAME vs MONIKER (CR 2.7.3): "Bravo, Star of the Show" has moniker "Bravo" — moniker-referencing effects hit all Bravos, but NAMING a card (Censor) matches the exact name only, whole-word, case-insensitive ("Blazing Aether" ≠ "Trailblazing Aether", 2.7.5); a moniker is not a name (2.7.3c). Deckbuilding legality runs on SUPERTYPES (CR 1.1.3): card-pool must be a subset of the hero's supertypes; Generic = no supertypes; hybrid "A / B" cards fit if either set fits (2.14.1b). Classes (2.11.6a): Adjudicator, Assassin, Bard, Brute, Guardian, Illusionist, Mechanologist, Merchant, Necromancer, Ninja, Pirate, Ranger, Runeblade, Shapeshifter, Thief, Warrior, Wizard. Talents (2.11.6b): Chaos, Draconic, Earth, Elemental, Ice, Light, Lightning, Mystic, Revered, Reviled, Royal, Shadow. Subtypes: functional ones add rules ((1H),(2H),Ally,Aura,Item,Landmark…, 2.10.6a); weapon-type subtypes (Sword, Dagger…) are non-functional.
