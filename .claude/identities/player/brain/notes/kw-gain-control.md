**Gain (control)** — effect keyword (CR 8.5.35).

Gain is a discrete effect. To gain control of an object, that player is considered to control the object instead of its previous controller. 8.5.35a If the object is equipped, the object will be equipped by the new controller as part of the gain event. If the object cannot be equipped this way, the effect fails.
