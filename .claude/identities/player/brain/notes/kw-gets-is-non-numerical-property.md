**Gets / Is (non-numerical property)** — effect keyword (CR 8.5.9).

Gets/is is a continuous effect, referred to as modify. To modify an object, the object gains the specified object properties.[2] Note: From Cards printed before 2023 have also used the keyword "gains/gain" and "has/have." 8.5.9a If the effect specifies the object gains a property, the object gains that property in addition to any existing properties it already has.
