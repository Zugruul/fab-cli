All 14 type keywords (CR 8.1): ACTION — needs empty stack + not-during-combat (except Resolution Step) + 1 action point; played "as an instant" it stays an action but loses those constraints and the AP cost (8.1.1d). ATTACK REACTION — attacker only, Reaction Step. DEFENSE REACTION — defender only, Reaction Step, becomes a defending card on resolve. INSTANT — any time you have priority. EQUIPMENT — equipped at setup, can be declared defending (8.1.4b). HERO / TOKEN (arena-only, ceases if it leaves) / WEAPON / RESOURCE (unplayable) / BLOCK (unplayable) / MACRO (not a card). MENTOR — deck-legal only with a young-subtype hero (8.1.10a). DEMI-HERO — becomes your hero if you control none, else cleared (8.1.11b). COMPANION — arena-card.
