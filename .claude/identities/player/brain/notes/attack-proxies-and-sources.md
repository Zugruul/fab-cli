Weapon/ability attacks create an ATTACK-PROXY — a non-card object extending its source (CR 1.4.3). The proxy inherits the source's properties (name, types, {p}) but NOT its activated/resolution abilities (1.4.3a — Edge of Autumn's printed "go again" is part of the ability so the proxy has it; Burn Them All's resolution 'go again' is not inherited). Buffs to the SOURCE flow to the proxy (Ironsong Determination on weapon → proxy has +1{p}); effects on the proxy don't touch the source, and "non-attack card" effects (Fog Down) miss proxies entirely (1.4.3d). Re-attacking with the same weapon kills the old proxy (1.4.3c). Attack-targets: declared when put on stack, must be an opposing living (or Spectra-style attackable) object (1.4.5); changing target between attacks does NOT close the chain (1.4.5b).
