Go again is a special resolution ability meaning "Gain 1 action point" (CR 8.3.5). On an attack, the action point is granted at the start of the Resolution Step (CR 8.3.5b); on a non-attack, after other resolution effects (CR 8.3.5a). Because you only get 1 action point per turn by default ([[action-points-and-go-again]]), go again is what lets a turn keep going — the engine behind multi-attack turns ([[tempo-and-go-again-chains]]) and long combat chains ([[combat-chain-and-chain-links]]). An object can only have one go again; a second instance fails (CR 8.3.5c). Many cards have it conditionally ("if X, this gets go again"). Sequencing: spend go-again cards first, end on a non-go-again action. This is the single most gameplay-central keyword. Verify any card's exact wording in [[card-corpus-and-search]].
